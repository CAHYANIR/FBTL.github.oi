Dasar Pemrograman Web

Dasar Pemrograman Web atau bisa disingkat dengan (DPW) merupakan sebuah mata kuliah yang mempelajari tentang dasar dari pembuatan sebuah web. Di mata kuliah DPW ini kita akan diajarkan untuk membuat dan mendesain sebuah halaman web dan membuat form. Untuk membuat sebuah web dan form kita akan diajarkan untuk menggunakan kode HTML, sedangkan untuk mendesain web sendiri kita akan diajarkan untuk menggunakan kode CSS dam untuk men-validasi form kita diajarkan untuk menggunakan kode JS).

Di mata kuliah DPW ini kebetulan kelompok kami mendapat sebuah tugas akhir (TA), untuk membuat sebuah aplikasi dengan tema Philanthropy Organization untuk tujuan keberlangsungan pendidikan. Setelah kami berdiskusi akhirnya memutuskan untuk memilih "From Book To LIfe" sebagai judul untuk TA kelompok kami. Di bawah ini merupakan nama-nama dari anggota kelompok kami beserta tugasnya.
